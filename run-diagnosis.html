诊断指纹信息显示问题

问题描述

安全仪表板无法显示指纹哈希值，即使主测试页面已经采集了指纹信息。

诊断步骤

  1. 检查localStorage中是否有共享数据
  2. 检查共享数据中是否有指纹信息
  3. 检查指纹信息中是否有哈希值
  4. 检查页面元素是否正确
  5. 检查JavaScript函数是否正常工作

快速修复建议

  1. 检查主测试页面: 确保访问的是 主测试页面 并看到控制台显示"指纹信息已收集"
  2. 强制刷新: 按 Ctrl+F5 清除浏览器缓存
  3. 检查localStorage: 按F12 → Application → Local Storage 查看 ks_shared_trajectory_data
  4. 运行诊断: 点击上面的"运行诊断"按钮
